# Bayesian Multifractal Image Segmentation
Matlab implementation for the multifractal analysis based image segmentation method.

<p align="center">
  <a href="https://sites.google.com/view/kleonlopez/home">Kareth León</a>,

  <a href="https://www.irit.fr/~Herwig.Wendt/index.html">Herwig Wendt</a>

  <img src="https://github.com/kareth-leon/MFA_Seg/blob/main/image/img_results.png">
  </p>

This work introduces an unsupervised Bayesian multifractal segmentation method to model and segment multifractal textures by jointly estimating the multifractal parameters and labels on
images, at the pixel-level.

## External Toolbox
Bayesian univariate and multivariate models and estimators for (c1,c2)
